Interest Calculator
Github Repository

The Interest Calculator is a SPA that allows the user to keep track of interest growth. Added features include filtering through monthly and yearly compounding periods and intial amounts.

Tech Stack:

REACT | BOOTSTRAP | NODE.JS | React Router | CSS3 |

  • Using React to manage Application State
  • The user is able to enter the monetary amount and loan duration by using an html slider element
  • The calculated values is automatically updated as the slider is used - without requiring any further interaction by the user
